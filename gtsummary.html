gtsummary

Using gtsummary package to create publication ready tables

Authors

Hannah Bowley

Micaela Lembo

What is gtsummary?

  • R tool that creates publication ready tables

  • Summarizes data sets, regression models and other analysis

  • Highly customizable

Install gtsummary package

#install.packages("gtsummary")

Load in libraries

library(gtsummary)
library(gapminder)
library(tidyverse)

Create Basic gtsummary table

table1 <- 
  gapminder %>%
  filter(year == 2007) %>% 
  select( gdpPercap, lifeExp, pop ) %>%

Customization option: label(), modify_header(), bold_labels()

table1 <- 
  gapminder %>%
  filter(year == 2007) %>% 
  select( gdpPercap, lifeExp, pop ,continent ) %>% 
  tbl_summary(
    by = continent, 
    label = list (gdpPercap ~ "GDP per capita",
                  lifeExp ~ "Life expectancy",
                  pop ~ "Population"
                  ) 
    ) %>% 
  modify_header(label = "**Variables**")  %>% 
  bold_labels()

Customization change default statistics:

  • statistics()
  • add p-value: add_p()
table1 <- 
  gapminder %>%
  filter(year == 2007) %>% 
  select( gdpPercap, lifeExp, pop ,continent ) %>% 
  tbl_summary(
    by = continent, 
    label = list (
      gdpPercap ~ "GDP per capita",
      lifeExp ~ "Life expectancy",
      pop ~ "Population"
      ) ,
    statistic = list( gdpPercap ~ "{mean}")
    ) %>% 
  modify_header(label = "**Variables**") %>% 
  add_p()

Customization to add a column with overall statistics: add_overall()

table1 <- 
  gapminder %>%
  filter(year == 2007) %>% 
  select( gdpPercap, lifeExp, pop ,continent ) %>% 
  tbl_summary(
    by = continent, 
    label = list (
      gdpPercap ~ "GDP per capita",
      lifeExp ~ "Life expectancy",
      pop ~ "Population"
      ) ,
    statistic = list( gdpPercap ~ "{mean}")
    ) %>% 
  modify_header(label = "**Variables**") %>% 
  add_overall( col_label = "**Global**",last = TRUE ) %>% 
  add_p()

Using gtsummary for regression models

mod1 <- glm(lifeExp ~ year + continent + continent*year, data = gapminder)

table2 <- tbl_regression(mod1) %>% 
  bold_labels() %>% 
  bold_p() %>% 
  modify_caption("Table 2: Regression results for Life Expectancy")
